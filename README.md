# BANCA

> Simulateur de trading médiéval — convertir et faire prospérer sa fortune.

---

## Aperçu

**Banca** est une interface interactive inspirée des pratiques bancaires de la Renaissance.  
Le projet met en scène un environnement immersif où l’utilisateur peut manipuler des devises historiques, interagir avec différentes interfaces et explorer une logique économique simplifiée.

L’accent est mis sur :

- une **direction artistique cohérente**
- une **interface immersive type livre**
- une **architecture front-end propre et évolutive**

---

## Fonctionnalités

- **Marché**
  - Consultation et conversion de monnaies historiques

- **Boutique** (EN COURS)
  - Catalogue visuel d’éléments (UI ready)

- **Paramètres** (EN COURS)
  - Gestion du son (toggle + volume)
  - Formulaire utilisateur

- **Ambiance**
  - Personnages animés en arrière-plan (idle animations CSS)
  - UI inspirée des manuscrits anciens
